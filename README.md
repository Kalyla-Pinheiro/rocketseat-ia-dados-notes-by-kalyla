## About the Course

The **AI for Data** course by Rocketseat aims to empower professionals to apply **Generative AI** to solve complex data challenges.

Throughout the course, topics such as **Prompt Engineering**, **Retrieval-Augmented Generation (RAG)**, and the use of **Generative AI to enhance classical Machine Learning algorithms** are explored using modern data tools.

---

## Course Structure

### Module 1 — AI for Data: Content and Practice

This module introduces the fundamentals and initial practices of **AI applied to Data**, focusing on modern and applicable techniques.

**Topics covered:**
- Prompt Engineering  
  - Role Playing  
  - Chain of Thought  
  - Few-shot  
  - System Message  
- Retrieval-Augmented Generation (RAG)  
  - Implementation using **N8n** and **Python**  
  - Strategies to reduce AI hallucinations  
- Optimization of classical **Machine Learning** algorithms  
  - Use of **synthetic data** generated by **Gemini**

## Sobre o Curso

O curso **IA para Dados**, da Rocketseat, tem como objetivo capacitar profissionais a aplicar **IA Generativa** para resolver desafios complexos relacionados a dados.

Ao longo do curso, são abordados temas como **Engenharia de Prompt**, **Geração Aumentada de Recuperação (RAG)** e o uso de **IA Generativa para aprimorar algoritmos clássicos de Machine Learning**, utilizando ferramentas modernas do ecossistema de dados.

---

## Estrutura do Curso

### Módulo 1 — Conteúdo e Prática de IA para Dados

Neste módulo são apresentados os fundamentos e práticas iniciais de **IA aplicada a Dados**, com foco em técnicas modernas e aplicáveis.

**Conteúdos abordados:**
- Engenharia de Prompt  
  - Role Playing  
  - Chain of Thought  
  - Few-shot  
  - System Message  
- Geração Aumentada de Recuperação (RAG)  
  - Implementação com **N8n** e **Python**  
  - Redução de alucinações da IA  
- Otimização de algoritmos clássicos de **Machine Learning**  
  - Uso de **dados sintéticos** gerados pelo **Gemini**
